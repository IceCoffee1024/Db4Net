# Product Notes

This directory is for product-side notes that are not part of the public VitePress documentation site.

Use this directory for:

- PRDs and early requirement drafts.
- Product prototypes and interaction notes.
- User stories and product discovery notes.

When a product idea becomes an implementation target, move or summarize the engineering-ready content into `docs/superpowers/specs/` or `docs/superpowers/plans/`.
